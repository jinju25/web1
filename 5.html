    2. css를 사용해야 하는 이유 (검색엔진)
    글씨 색상 바꾸기 color: rgb(0, 0, 0)
    style이라는 속성을 썼으면 그 속성의 값을 웹브라우저는 css의 문법에 따라 해석

    같은 그룹으로 묶고싶은 대상을 class 값을 갖는 태그를 만들고 선택자 앞에 . 을 붙여 class 값을 갖는 태그를 선택자로 지정하는것을 알려준다.
    class라고 하는 속성의 값은 여러 개의 값이 들어올 수 있고
    띄어쓰기로 구분한다.

    id값은 중복되어서는 안된다.
    id값은 유일무이하고 구체적이기 때문에 우선순위가 높다.
    #ID선택자 >> .으로 표현된 class선택자 >> 태그 선택자
